## 2. 산술 연산자와 합성 연산자

### 산술 연산자
**NUMBER,DATE** 자료형에 대해서 적용.
alias을 같이 이용하자
```sql
SELECT height-weight as '키-몸무게'
FROM table1;
```

## 합성 연산자
```sql
SELECT P_NAME || '선수,' || HEIGHT || 'cm.'  as '선수 키 정보'
FROM TABLE1;
/*선수 키 정보
--------------
A 선수, 190 cm.
B 선수, 182 cm.
*/
```
